Welcome in servlet 3 api demo

Following links should work:

  • [/index.html] - this page = default toplevel welcome page
  • [/jsphello.jsp] - simple toplevel JSP page
  • [/fragment] - welcome page defined in the web-fragment.xml
  • [/fragment/main.html] - fragment's welcome page explicitly
  • [/main.html] - just to remind that fragment can add resources anywhere, including toplevel (it does not have any dedicated subcontext)
  • [/fragment/hello] - servlet from fragment mapped by annotation
  • [/fragment/oldway] - servlet from fragment mapped by the old way declaration in web-fragment.xml

Following links should NOT work:

  • [/WEB-INF/hidden.txt] - toplevel's protected area
  • [/META-INF/hidden.txt] - fragment's protected area
